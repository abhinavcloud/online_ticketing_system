# Browse Service – Lambda Console Test Events (All Endpoints)

These events are ready to paste into **AWS Lambda Console → Test → Configure test event**.

They use **HTTP API (v2) style** payloads (`version: 2.0`, `rawPath`, `requestContext.http.method`).

## Replace these placeholders
- LOCATION_ID: your UUID (example: `2cef468d-eee9-4a96-9d5a-7fda95910235`)
- VENUE_ID: your UUID (example: `533d8812-6dd9-48f4-80b8-3d82f06117fc`)
- PERFORMER_ID: your UUID
- EVENT_ID: your UUID

---

## 1) GET /v1/location

```json
{
  "version": "2.0",
  "rawPath": "/v1/location",
  "queryStringParameters": {
    "page": "1",
    "pageSize": "10"
  },
  "requestContext": {
    "http": {
      "method": "GET",
      "path": "/v1/location"
    }
  }
}
```
